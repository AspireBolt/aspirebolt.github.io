PREEMPTIVE SHORTEST REMAINING TIME FIRST

  • Category: Software Development
  • Project date: Januray 29,2024

Details

In the world with operating systems, effective CPU scheduling is essential for maximizing system efficiency. CPU scheduling is the deliberate distribution of the CPU's processing time among multiple processes, enabling the execution of one while others may be temporarily deferred due to factors such as I/O operations. The use of CPU scheduling aims to improve system speed, fairness, and efficiency. For this project, we are tasked to create a scheduling program that makes you see how this kind of algorithm will work. For this project, we are tasked to create the preemptive shortest remaining time first (SRTF) algorithm. Within this project, the following tools used is going to be used for this exercise.

Software and Hardware Used:
